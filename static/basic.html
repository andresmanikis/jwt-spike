Basic JWT example

This is a basic example. You have two buttons.

The first button makes a GET request to an authorized resource. To get there you will need a valid JWT token. For example, a valid one is this one and it holds a name inside its payload. Once you press the button you should see the response from the server in the console. Try changing the token or corrupting it (just deleting or changing a char will do the trick) and see what happens.

The second button makes a GET request to an open resource. You can get there whenever you like with any token.

Open your console to see the result of your actions.
